# Retail Pulse Analytics

## Project Overview
Customer Lifetime Value (CLV) Dashboard built using Power BI.

## Dashboard Metrics
- Total Revenue
- Total Customers
- Total Orders
- Average Revenue per Customer

## Customer Segments
- VIP
- High Value
- Medium Value
- Low Value

## Dashboard Pages

### Page 1: Executive Dashboard
- KPI Cards
- Revenue by Segment
- Customer Distribution
- Revenue Share Analysis
- Interactive Slicer

### Page 2: Customer Segment Analysis
- Revenue Comparison
- Orders Comparison
- Segment Summary Table
- Business Insights

## Tools Used
- Power BI
- Python
- Snowflake
- Git & GitHub
